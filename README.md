# CrediNews AI 📰🤖

CrediNews AI is an **AI-powered real-time news credibility checker** that helps identify whether a news article is real or fake.

The system uses **machine learning and real-time news verification APIs** to analyze credibility.

---

## 🚀 Features

* Detect fake vs real news using AI
* Credibility score prediction
* React frontend interface
* Flask backend API
* News verification using external APIs

---

## 🧠 How It Works

1. User enters a news headline or article.
2. The text is sent to the backend API.
3. The machine learning model analyzes the text.
4. The system checks external news sources.
5. The final credibility result is returned.

## ⚙️ Installation

Clone the repository:

git clone https://github.com/Pratyush-Kulshreshtha-35/credinews-ai.git

Backend setup:

cd backend
pip install -r requirements.txt
python app.py

Frontend setup:

cd frontend
npm install
npm start
